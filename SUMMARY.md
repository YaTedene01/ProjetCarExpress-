## 🎨 Design Livré

### Palette de Couleurs
```
🔴 Rouge (#D40511)      → Couleur primaire
🟡 Jaune (#FFCC00)      → Accent
⚫ Noir (#111111)        → Texte principal
⚪ Gris (#5f5750)        → Texte secondaire
🟢 Vert (#1a7a2e)       → Success
```

### Typographie
```
Display:  Space Grotesk (titres, headings)
Body:     Plus Jakarta Sans (texte, interfaces)
Mono:     JetBrains Mono (code)
```

### Espacement
```
xs: 4px    | sm: 8px    | md: 16px   | lg: 24px
xl: 32px   | 2xl: 40px  | 3xl: 60px
```

### Animations
```
Float (3s)   | Slide In (0.5s) | Fade In (0.5s) | Pulse (2s)
```
